# El Barato v0.1.13

## Cambios

- El desktop ahora fuerza una actualizacion del catalogo local cuando detecta una version vieja, para bajar tambien los precios `x3`, `x6` y `x12`.
- El bootstrap local ahora deja claro que tambien descarga la lista de precios tier.
- El buscador del Punto de Venta mantiene el foco despues de agregar un producto, para seguir escribiendo sin volver a hacer clic.
